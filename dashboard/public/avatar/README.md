# Avatar PNG Layers

Place PNG files (512×512, transparent background) here.

## Required files
- `base.png` — body + face
- `clothing/winter.png`, `warm.png`, `mild.png`, `hot.png`, `rain.png`
- `accessories/umbrella.png`, `sunglasses.png`, `scarf.png`
- `expressions/normal.png`, `tired.png`, `energetic.png`
- `effects/yawn.png`, `spark.png`, `halo.png`

## Style: pixel art or flat illustration, lo-fi aesthetic, 512×512 canvas
